# commerceml

Сделать по аналогии с webref.ru (только репозиторий лучше)

Файлы:
- import.xml - содержит информацию о товарах, свойствах, категориях
- offers.xml - содержит цены, остатки, характеристики

# import.xml

Состоит из 2-ух групп тегов:
- Классификатор
- Каталог

## <Классификатор>

- <Ид>
- <Наименование>
- <Владелец>
	- <Ид>
	- <Наименование>
	- <ОфициальноеНаименование>
	- <ИНН>
	- <КПП>
- <Группы>
	- <Группа>
		- <Ид>
		- <Наименование>
		- <Группы>
			- ...
- <Свойства>
	- <Свойство>
		- <Ид>
		- <Наименование>
		- <Варианты значений>
			- <ИдЗначения>
			- <Значение>

## <Каталог>

		- <Ид>
		- <Артикул>
		- <Наименование>
		- <БазоваяЕдиница>
		- <ПолноеНаименование>
		- <Группы>
			- <Группа>
				- <Ид>
		- <Описание>
		- <ЗначенияСвойств>
			- <ЗначенияСвойства>

				- <Значение>
		- <СтавкиНалогов>
